SQL(Structured Query Language): 데이터 베이스를 직접적으로 엑세스 할 수 있는 언어

DBMS : 데이터베이스 관리 시스템
DDL(Data Definition) : 데이터를 정의(생성, 삭제)
DML (Data Maninpulation): 데이터 조작 (수정, 업데이트 , 삭제, 추가 등)
DCL(Data Control): 데이터 접근 권한, 커밋 , 롤백(transaction control)
